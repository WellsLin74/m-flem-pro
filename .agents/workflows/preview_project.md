# Preview Project

This workflow explains how to start the development preview server for the M-FLEM Pro NextJS application.

1. Start the NextJS development server:
// turbo
```bash
npm run dev
```

2. Open your browser and navigate to `http://localhost:9002` (or the port specified in the terminal output) to test the application interactively.
